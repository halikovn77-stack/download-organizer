# 📁 Органайзер загрузок

[![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Автоматическая сортировка файлов в папке `Загрузки` по категориям. Программа работает в фоновом режиме и автоматически раскладывает новые файлы по папкам: изображения, документы, архивы, видео, музыка.

## ✨ Возможности

- 🚀 **Автоматическая сортировка** — следит за папкой и сортирует новые файлы
- 📁 **Защита от перезаписи** — если файл с таким именем уже существует, добавляет номер: `cat.jpg` → `cat_1.jpg`
- ⚙️ **Гибкие настройки** — правила сортировки можно менять в `config.json` без редактирования кода
- 📝 **Логирование** — все действия записываются в файл `organizer.log`
- 🖥️ **Кроссплатформенность** — работает на Windows, macOS и Linux

## 📦 Установка

### 1. Проверьте, что установлен Python 3.6 или выше

```bash
python --version
# или
python3 --version
```

Если Python не установлен — скачайте с python.org

### 2. Скачайте проект

```bash
git clone https://github.com/ВАШ_НИК/download-organizer.git
cd download-organizer
```

Или скачайте ZIP-архив со страницы репозитория.

### 3. Установите зависимости

```bash
pip install watchdog
```

## ⚙️ Настройка

### Правила сортировки

Скопируйте файл `config.example.json` в `config.json` и отредактируйте под свои нужды:

```bash
cp config.example.json config.json
```
Пример `config.json`:

```json
{
    "Изображения": [".jpg", ".jpeg", ".png", ".gif", ".bmp", ".webp"],
    "Документы": [".pdf", ".docx", ".txt", ".xlsx", ".pptx", ".rtf"],
    "Архивы": [".zip", ".7z", ".rar", ".tar", ".gz"],
    "Видео": [".mp4", ".mkv", ".avi", ".mov", ".wmv"],
    "Музыка": [".mp3", ".wav", ".flac", ".aac"]
}
```

Вы можете:

 -  Добавлять новые типы файлов
 -  Удалять ненужные
 -  Менять названия папок

### Путь к папке Загрузки

По умолчанию программа следит за папкой `~/Download`s. Если вы хотите изменить путь — отредактируйте переменную `my_downloads` в файле `organizer.py`:

```python
my_downloads = os.path.expanduser("~/Downloads")  # Замените на свой путь
```

## 🚀 Запуск

### Обычный запуск

```bash
python organizer.py
```

### Автоматический режим

Программа запустится в фоне и будет следить за новыми файлами. Для остановки нажмите `Ctrl+C`.

### Проверка логов

Все действия записываются в файл `organizer.log`. Вы можете просмотреть его в любом текстовом редакторе.

## 🐛 Известные проблемы и их решение

### Файл не перемещается сразу после скачивания

Некоторые браузеры создают временные файлы (.crdownload, .part). Программа игнорирует их и дожидается полной загрузки.

### Ошибка "Permission denied"

Убедитесь, что у программы есть доступ к папке Загрузки. На некоторых системах может потребоваться запуск с правами администратора.
